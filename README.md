# grb-lightcurve-modelling

Codes for GRB light-curve modelling and fitting developed for my Master's thesis.

The repository is organized into two main folders:

- `extended_emission/`: codes for the modelling and fitting of short GRBs with extended emission (EE), based on a top-hat high-latitude emission model.
- `plateau/`: codes for the modelling and fitting of plateau GRBs, including structured-jet high-latitude emission, forward-shock modelling, and combined fitting.

This repository is intended as a compact archive of the main codes used in the thesis work.
