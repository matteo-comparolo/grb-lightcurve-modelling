# Extended-emission GRBs

This folder contains the main codes used for the modelling and fitting of short GRBs with extended emission (EE), based on a top-hat high-latitude emission model.

Files included:
- `fit_hle_xrt_tophat.py`: fitting script for the X-ray light curves
- `hle_tophat.py`: top-hat high-latitude emission model
- `params_tophat.yaml`: example parameter/configuration file
